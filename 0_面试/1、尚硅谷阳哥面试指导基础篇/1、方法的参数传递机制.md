# 一、传值和传引用

基本类型是传值：即传值之后原值不变（传的是复印件，原件不变）；

引用类型是传引用：即传的是“指针” （内存地址）；

String是**特殊**的引用类型；（String的值是在字符串常量池中，如果赋新的值，如果有则复用，如果没有就新建）
