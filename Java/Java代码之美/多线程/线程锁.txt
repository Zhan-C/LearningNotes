1、什么是线程锁，

	就是讲一个线程运行时将他锁起来在他执行之后执行下一次操作

2、为什么需要线程锁
	
	当我们在使用同一个Runnable对象，和线程对象，以及其他的线程调度同一个对象时，
	如果两个线程同时去修改一个数据则会发生错误，这也是我们说的线程安全的问题，
	所以为了防止这种情况我们需要锁定他

	使用方法：

	在执行过程中加入

        @Override
    	public void run() {
        	for (int i = 0; i < 3; i++) {
			//这里的参数可以随便写，但通常用常量来代替，因为如果创建对象的或者其他的时候发生改变线程锁就不同步了
            		synchronized (""){
				//Java语句块
			}
           	 }
        }


	然后我们再来使用线程就可以了
